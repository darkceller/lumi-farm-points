## 📖 專案概述 (Project Overview)

本專案為實踐「光農共生圈」理念，專為樸爪農園與綠手臂生活實驗室量身打造的\*\*「去中心化社群碳積分系統」\*\*。

為了減輕「半農半 X」狀態下的行政管理負擔，本系統採用極簡的 Low-Code 架構。將志工在土地上流汗的勞動，透過數位化轉化為流動的社群信用（數位菌絲）。志工只需透過手機掃描現場 QR Code 即可完成照片存證與時數登記；管理端則整合 Google 試算表，實現一鍵核准、自動寄信與留白區認領扣點的無縫管理。

-----

## ✨ 核心特色 (Core Features)

1.  **極速載入體驗**：前端採純 HTML/CSS/JS 撰寫，零外部框架（Zero Dependencies），完美適應山區農園不穩定的網路環境。
2.  **生態系加權引擎**：內建四大勞動維度（常規紀錄、技術繁衍、高強度建設、社群推廣），精準賦予 1.0x 至 1.5x 的碳積分加權，彰顯保種與生態觀察的價值。
3.  **無痛防弊與存證**：強制調用手機相機進行現場拍攝，照片自動轉碼 (Base64) 存入 Google Drive，確保勞動紀錄的真實性。
4.  **一鍵批次審核**：後台結合 Google Apps Script (GAS) 自訂選單，管理員只需切換下拉狀態，即可一鍵批次寄送帶有「光農合作社群」專屬署名的核准通知信。
5.  **夢想留白區經濟循環**：內建扣點與期限追蹤機制，讓碳積分具備實質兌換價值（如認領 1-3 坪實驗基地），完整閉環社群經濟。
6.  **社群無縫導流**：整合 Substack 電子報訂閱模組，將登記流程與社群資訊中心（光農．食光周報）完美綁定。

## 🛠️ 技術架構 (Tech Stack)

  * **前端介面 (Frontend)**: HTML5, CSS3, Vanilla JavaScript (Fetch API, FileReader)
  * **雲端運算 (Serverless API)**: Google Apps Script (GAS)
  * **資料庫與管理端 (Database & Admin)**: Google Workspace (Google Sheets)
  * **檔案儲存 (Storage)**: Google Drive API
  * **通訊與社群 (Comms)**: Gmail API, Substack Embed
  * **版本控制與代管 (Hosting)**: Git, GitHub Pages

## 🔮 未來優化藍圖 (Roadmap)

為確保系統隨著中壢老城區與在地農場的社群擴大而具備擴充性，預計於後續階段評估導入以下功能：

  - [ ] **前端影像壓縮 (Canvas API)**：在手機端先壓縮照片再轉碼 Base64，避免高畫素手機造成網路傳輸超時 (Timeout)。
  - [ ] **GPS 地理圍欄防弊 (Geolocation)**：調用瀏覽器定位權限，檢核簽到座標是否位於合作農園半徑內。
  - [ ] **家庭帳戶綁定 (Family\_ID)**：新增前端群組代號綁定機制，便於同一家庭成員合併計算點數以共同認領大型留白區。
  - [ ] **總體碳積分儀表板 (Dashboard)**：在前端撈取並匯總全站點數，展示「本月共減少多少碳足跡 / 累積多少勞動時數」，強化社群共榮感。
